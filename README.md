# fullstack-open

Solutions and notes for the [Full Stack Open](https://fullstackopen.com/en/) course
by the University of Helsinki. This is a rigorous, project-based curriculum covering the
full JavaScript stack from React fundamentals through to TypeScript, GraphQL, and CI/CD.

I'm working through this course to deepen my full-stack foundations alongside
real-world projects. All solutions are my own, committed only once working.
